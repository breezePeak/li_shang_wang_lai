# 安全说明

如果你发现的是普通 Bug，请直接提 Issue。

如果你发现的是下面这类问题，请不要公开贴出细节，建议私下联系仓库维护者：

- 账号凭据泄露风险
- Cookie / Token / API key 泄露
- 可以误操作他人账号的逻辑缺陷
- 会导致批量误发、批量误赞、状态错乱的大问题
- 可能触发平台风控或合规风险的实现细节

## 不要公开提交的信息

- `.env` 内容
- `config/local.json`
- 浏览器登录态
- `data/` 目录下的本地数据库
- 带有个人信息的原始互动记录

## 报告内容建议

请尽量提供：

- 问题类型
- 影响范围
- 复现步骤
- 是否需要真实账号才能复现
- 你建议的修复方向

## 项目边界

这个项目是本地自动化工具，不提供线上托管服务，也不会主动收集你的账号数据。

但它会与真实平台页面和接口交互，所以你仍然需要自己承担本地环境、账号使用和平台规则变化带来的风险。
