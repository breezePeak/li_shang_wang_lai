# Changelog

本项目遵循“先记录重要变化，再逐步细化”的维护方式。

## 2026-06-13

### Added

- 支持按最近几小时而不是仅按天来筛选扫描与回评执行窗口
- Web 控制台增加更明确的异常回评人工处理入口
- 增加开源仓库所需的 `LICENSE`、`CONTRIBUTING`、`SECURITY` 等文档

### Changed

- 评论提交成功判定改为严格依赖真实 `comment/publish` API 成功响应
- 已忽略 `skipped` 和已成功 `succeeded` 的回评状态现在会被锁定，不能再被前端或接口修改
- 扫描再次遇到已忽略评论时，不再自动恢复为 `pending`

### Fixed

- 修复“接口看起来成功，但其实没有真实评论结果”仍被误判成功的问题
- 修复忽略后的评论被后续扫描重新拉回执行队列的问题
